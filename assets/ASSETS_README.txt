ASSETS REQUIRED FOR PRODUCTION
==============================

1. favicon.ico
   - Create a 32x32 or 48x48 ICO file
   - Can use the provided favicon.svg as a base
   - Place at: /assets/favicon.ico

2. og-image.jpg
   - Recommended size: 1200x630px
   - Should include:
     * "Claim Tools AI" branding
     * Value proposition: "$2,156 in tools for $997"
     * Professional design matching site aesthetic
   - Place at: /assets/og-image.jpg

3. Product Icons (Optional)
   - SVG icons for each product category
   - Place in: /assets/icons/
   - Currently using emoji as placeholders

NOTE: The site will work without these assets, but they're important for:
- Browser tab branding (favicon)
- Social media sharing previews (og-image)
- Professional appearance
