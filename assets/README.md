# Assets Directory

This directory contains the following assets:

## Required Files

1. **favicon.ico** - Website favicon (16x16, 32x32, or 48x48 ICO format)
2. **og-image.jpg** - Open Graph social sharing image (1200x630px recommended)

## Icons

The `/assets/icons/` directory can contain SVG icons for products if needed.

## Notes

- Replace `favicon.ico` with your actual favicon
- Replace `og-image.jpg` with a professional 1200x630px image showing your branding
- The OG image should include: "Claim Tools AI" branding, value proposition, and visual appeal
